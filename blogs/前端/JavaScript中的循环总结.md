## for

> for是最基础的循环，常用于数组的循环遍历

1. 循环数组

    ```javascript
    let arr = [1,2,3]
    for(let i = 0; i<arr.length; i++){
    	console.log(i,arr[i])
    }
    //结果
    0,1
    1,2
    2,3
    ```
